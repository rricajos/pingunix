# 103.2 - Procesar flujos de texto con filtros

## Peso: 2

## Objetivo del examen
Aplicar filtros a flujos de texto. Enviar archivos de texto y flujos de salida a traves de utilidades de filtrado de texto para modificar la salida utilizando comandos UNIX estandar que se encuentran en el paquete GNU textutils.

## Conocimientos clave
- Enviar archivos de texto y flujos de salida a traves de filtros de texto para modificar la salida
- Utilizar comandos UNIX estandar del paquete GNU textutils
- Conocer los comandos de filtrado y sus opciones principales
