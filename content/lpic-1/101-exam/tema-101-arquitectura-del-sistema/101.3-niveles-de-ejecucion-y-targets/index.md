# 101.3 - Niveles de Ejecucion y Targets de Systemd

## Peso: 3

## Objetivo del examen
Establecer el nivel de ejecucion o target de arranque del sistema y cambiar entre niveles de ejecucion / targets, incluyendo el modo de usuario unico.

## Conocimientos clave
- Establecer el nivel de ejecucion o target por defecto
- Cambiar entre niveles de ejecucion / targets incluyendo modo monousuario
- Apagar y reiniciar desde la linea de comandos
- Alertar a los usuarios antes de cambiar de nivel de ejecucion / target
- Terminar procesos correctamente
- Conocimiento de acpid

## Archivos, terminos y utilidades
- `/etc/inittab`
- `shutdown`, `init`, `telinit`
- `systemctl`
- `/etc/init.d/`, `/etc/rc.d/`
- `wall`
